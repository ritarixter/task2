## Сборка проекта

Чтобы собрать и запустить проект на [localhost:3000](http://localhost:3000) выполните следующие команды в своей локальной директории:

- `$ git clone https://github.com/ritarixter/task2.git`
- `$ npm install`
- `$ npm start`
